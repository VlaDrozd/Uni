События объекта window

  • onbeforeunload – выполняет код JavaScript перед выгрузкой страницы, что позволяет предварительно выполнить необходимые действия;
  • onblur – выполняет код JavaScript при утрате фокуса окном или фреймом;
  • onerror - выполняет код JavaScript, когда загрузка документа приводит к ошибке;
  • onfocus - выполняет код JavaScript при получении фокуса окном или фреймом;
  • onhelp - выполняет код JavaScript при нажатии пользователем клавиши F1;
  • onload - выполняет код JavaScript, когда браузер завершает загрузку окна или всех фреймов в тэге FRAMESET;
  • onresize - выполняет код JavaScript, когда пользователь или сценарий изменяет размеры окна или фрейма;
  • onscroll - выполняет код JavaScript при прокрутке окна пользователем;
  • onunload - выполняет код JavaScript, когда пользователь завершает работу сдокументом.
